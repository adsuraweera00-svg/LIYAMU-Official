LIYAMU

A comprehensive literary platform connecting authors, readers, and creative minds

📚 Project Overview

LIYAMU is a full-stack web application designed to empower authors, readers, and creative professionals. The platform provides tools for publishing, discovering, and engaging with literary content.

🛠 Tech Stack

Frontend React • Vite • Tailwind CSS
Backend Node.js • Express • MongoDB
Authentication JWT • Secure Password Hashing
Storage Cloudinary • File Management

📁 Repository Structure

Backend: RESTful API with controllers, models, middleware, and authentication

Frontend: React SPA with routing, state management, and responsive UI

Features: Book publishing, user profiles, creative works, credits system, earnings tracking, and more
